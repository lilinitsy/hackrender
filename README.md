# hackrender
Hackrender in its current form is a conglomeration of animation methods that will eventually be merged to be used a 3D Nethack game.

### Features
Particle Engine

Newtonian Mechanics

### Planned Features
Cloth Simulation

Hair Sim

SPH Fluid Simulation
